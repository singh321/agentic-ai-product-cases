# AI Recruiter Agent

## Problem
Recruiters spend time screening and shortlisting candidates.

---

## Goal
Automate candidate screening using AI.

---

## Workflow

1. ingest resumes
2. extract skills
3. match with JD
4. rank candidates
5. suggest shortlist

---

## System

- resume parser
- LLM scoring
- ranking engine

---

## Evaluation

- match accuracy
- recruiter satisfaction

---

## Metrics

- time saved
- hiring speed
- shortlist quality

---

## Risks

- bias
- incorrect ranking
